# Hackaton - Backend

## 🐍 Suggested Pyhton version
```
3.11.*
```

## 📐 Setup

Linux, MacOs
``` bash
$ python3 -m venv venv
$ source venv/bin/activate
```

Windows
``` bash
$ py -m venv venv
$ source venv/scripts/activate
```

## 🚀 Dev

Before run the development server make sure to be running Azurite Blob Service on 127.0.0.1:10000

``` bash
$ func start
```

## 🧪 Test

``` bash
$ pytest tests
```
